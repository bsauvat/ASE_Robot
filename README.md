To use the DSL, you need to :
- clone this repository
```bash 
git clone https://github.com/bsauvat/ASE_Robot.git
```
- place yourself in the robot directory
```bash 
cd robot
```
- build the project
```bash 
npm run build:web
```
- run the server
```bash 
npm run serve
```
- open port 5173 :
